# 公文用語詳細對照表

本文件收錄公文用語的完整對照表，供撰寫時查閱。SKILL.md 已包含最常用的用語，此處提供更完整的補充。

## 一、引敘語完整對照

### 文句前端（開頭引述來文時）

| 用語 | 使用時機 | 範例 |
|------|----------|------|
| 奉 | 引敘上級機關或首長來文 | 「**奉**鈞長指示辦理公文訓練一案……」 |
| 准 | 引敘主管機關之規定或說明 | 「本案可否辦理留職停薪，**經函准**銓敘部解釋略以……」 |
| 據 | 引敘平行、下級機關或人民來文 | 「**據**本市各區公所查報結果……」 |

### 文句末端（引述完畢時）

| 用語 | 使用時機 |
|------|----------|
| 奉悉 | 回復上級機關來文或首長指示，於引敘完畢時使用 |
| 敬悉 | 回復平行機關來文，於引敘完畢時使用 |
| 已悉 | 回復下級機關公文，於引敘完畢時使用 |

### 表明依據

| 用語 | 使用時機 |
|------|----------|
| 遵照 | 對直屬上級機關表明依據 |
| 依照 | 對主管機關表明依據 |
| 依據、依照、根據 | 對平行、下級機關或屬員、人民來文時使用 |

### 續辦收束語（再度敘及時）

| 用語 | 使用時機 |
|------|----------|
| 諒蒙鈞察 | 對上級機關發文後續函時使用（亦作「收束語」） |
| 諒蒙、諒達 | 對主管機關 |
| 諒達、計達 | 對平行或下級機關 |

## 十、法律統一用字

| 正確 | 錯誤 | 說明 |
|------|------|------|
| 訂定 | 制定 | 法律以外之法規用「訂定」 |
| 制定 | 訂定 | 法律用「制定」 |
| 規定 | 規定 | 通用 |
| 設 | 設置 | 機關之設，用「設」 |
| 依照、依據 | 依照、依據 | 引用法條時使用 |
| 處○○以下罰鍰 | 處以○○以下罰鍰 | 「以」字多餘 |
| 科○○以下罰金 | 科以○○以下罰金 | 「以」字多餘 |
| 第○條之一 | 第○條之1 | 法條之一用中文 |
| 自公布日施行 | 自公佈日施行 | 法律用「公布」，一般事項用「公布」亦可 |
