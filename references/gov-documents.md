# 政府機關其他文件撰寫規範

本文件涵蓋政府機關常用但不屬於《公文程式條例》所列公文類別的文件格式。

## 一、開會通知單

### 使用時機
機關召開會議時，通知與會人員出席的文件。

### 結構

```
○○機關 開會通知單

發文日期：中華民國000年00月00日
發文字號：○○○字第0000000000號
速別：
密等及解密條件或保密期限：

開會事由：○○○○○○○○○○
開會時間：中華民國000年00月00日（星期○）上午/下午00時00分
開會地點：○○○○○○
主 持 人：○○○
聯 絡 人：○○○
聯絡電話：（00）0000-0000

出席者：○○○○○
列席者：○○○○○

備  註：
  一、……
  二、……

○○長 ○○○
```

### 撰寫要領
- 開會事由應簡明扼要
- 時間須明確到幾時幾分，並註明星期幾
- 地點應具體（含樓層、會議室名稱）
- 出席者與列席者分開列示
- 備註可載明：攜帶資料、停車資訊、是否供餐、請假方式等

## 二、會議紀錄

### 使用時機
記錄會議討論過程與決議事項。

### 結構

```
○○機關○○○會議紀錄

時  間：中華民國000年00月00日（星期○）上午/下午00時00分
地  點：○○○○○○
主  席：○○○                          記錄：○○○
出席人員：○○○、○○○、○○○（詳如簽到表）
列席人員：○○○、○○○
請假人員：○○○

壹、主席致詞：（略）

貳、報告事項：
  一、○○○○○○○○
    決定：○○○○○○

參、討論事項：
  案由一：○○○○○○○○（提案單位：○○○）
    說明：
      一、○○○○○○
      二、○○○○○○
    決議：○○○○○○

  案由二：○○○○○○○○
    說明：……
    決議：……

肆、臨時動議：（無）

伍、散會（○時○分）
```

### 撰寫要領
- 決議事項應明確具體，載明執行單位與期限
- 出席人員如多，可註「詳如簽到表」
- 報告事項與討論事項分開記載
- 討論事項應載明提案單位
- 發言內容以摘要方式記載，不必逐字記錄
- 會議紀錄經主席核定後，以函或書函送與會人員及相關單位

## 三、新聞稿

### 使用時機
機關對外發布新聞時使用。

### 結構

```
○○機關新聞稿

發稿日期：中華民國000年00月00日
發稿單位：○○○○
新聞聯絡人：○○○
聯絡電話：（00）0000-0000

【標題】○○○○○○○○○○○○

【副標題】○○○○○○○○○○（選用）

（本文：倒金字塔結構）

第一段：最重要的結論或訊息（Who, What, When, Where, Why）
第二段：補充說明、背景資訊
第三段：細節、數據、引述發言
第四段：後續規劃或呼籲
```

### 撰寫要領
- 採「倒金字塔」結構：最重要的訊息放最前面
- 標題應吸引目光且忠實反映內容
- 用語淺白，避免公文體（不用「查」「按」「爰」等公文用語）
- 第一段應能獨立使用，涵蓋 5W1H
- 引述首長發言時用直接引語
- 末段可附背景資訊或補充說明
- 新聞稿不是公文，不需要主旨/說明/辦法結構

## 四、施政報告 / 業務報告

### 結構
```
○○機關○○年度施政報告

壹、前言
貳、施政目標與重點
參、本年度重要施政成果
  一、○○業務
  二、○○業務
肆、未來展望
伍、結語
```

### 撰寫要領
- 有數據佐證（預算執行率、服務人次等）
- 成果與目標對照
- 問題與改進方向並陳

## 五、聘函 / 聘書

### 結構
```
○○機關 聘函（或聘書）

茲聘請 ○○○先生/女士 為本機關○○○○○（職稱或任務），
聘期自中華民國000年00月00日起至000年00月00日止。

此致
○○○先生/女士

○○機關
○○長 ○○○（簽字章或簽名蓋章）

中華民國000年00月00日
```

## 六、獎狀 / 資格證書

格式依各機關規定，通常包含：受獎人/受證人姓名、事由、依據、日期、機關首長署名蓋章。

## 共通格式提醒

- 用紙：A4，70～80 GSM
- 字體：標楷體
- 日期：使用中華民國紀年
- 分項標號：壹、貳、參（大綱層級）→ 一、二、三 → （一）（二）→ 1、2、3
