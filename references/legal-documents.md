# 法律文件撰寫規範

本文件涵蓋台灣常見法律相關文件的格式與撰寫規範。這些文件不屬於政府公文體系，但有其特定的格式要求與法律效力。

## 一、存證信函

### 使用時機
- 催告履行契約義務（如催繳租金、催還借款）
- 終止或解除契約的意思表示
- 主張權利或保全證據
- 通知對方特定事項並保留書面紀錄

### 結構

```
存證信函

寄件人：
姓名：○○○
地址：○○市○○區○○路○○號
電話：（00）0000-0000

收件人：
姓名：○○○
地址：○○市○○區○○路○○號

主旨：○○○○○○○○○○

說明：

一、（事實經過：時間、地點、雙方關係、契約依據）

二、（法律依據：引用相關法條）

三、（請求事項：限期催告、主張權利、解除契約等）

四、（法律效果警告：逾期未履行之法律後果）

此致
○○○先生/女士

寄件人：○○○（簽名蓋章）

中華民國000年00月00日
```

### 撰寫要領
- 至郵局購買存證信函專用紙（一式三份），或使用中華郵政網站下載格式
- 每頁20行、每行20格（手寫時），打字不受此限但建議整齊
- **事實要具體**：日期、金額、契約條款號數都要明確
- **法條要正確**：引用民法、消費者保護法等相關條文
- **催告要設期限**：如「請於函到7日內」
- **語氣堅定但不威脅**：陳述事實與法律效果，不使用恐嚇用語
- 以雙掛號郵寄，保留回執聯

### 常見催告用語
- 「請臺端於本函送達之日起○日內……」
- 「逾期未履行，本人將依法提起訴訟，屆時訴訟費用及律師費由臺端負擔」
- 「特此函告，以為存證」

## 二、合約書 / 契約書

### 基本結構

```
○○○○合約書（或契約書）

立合約書人：
甲方：○○○（以下簡稱「甲方」）
統一編號/身分證字號：
地址：
代表人：○○○

乙方：○○○（以下簡稱「乙方」）
統一編號/身分證字號：
地址：
代表人：○○○

緣甲乙雙方就○○○○事宜，經雙方協議合意，訂立本合約，條款如下：

第一條（合約標的）
○○○○○○

第二條（合約期間）
自中華民國000年00月00日起至000年00月00日止。

第三條（權利義務）
一、甲方應……
二、乙方應……

第四條（對價與付款）
○○○○○○

第五條（保密條款）
○○○○○○

第六條（違約責任）
○○○○○○

第七條（爭議處理）
本合約如有爭議，雙方同意先以協商方式解決；協商不成，合意以○○地方法院為第一審管轄法院。

第八條（其他約定）
本合約未盡事宜，依中華民國法律辦理。

本合約一式○份，由雙方各執○份為憑。

立合約書人

甲方：○○○（簽名蓋章）
代表人：○○○

乙方：○○○（簽名蓋章）
代表人：○○○

中華民國000年00月00日
```

### 撰寫要領
- 當事人資訊完整（名稱、統編/身分證、地址、代表人）
- 條款編號清楚，每條加標題
- 權利義務對等且具體
- 必備條款：標的、期間、對價、違約、管轄法院
- 金額同時以中文大寫與阿拉伯數字表示
- 日期使用中華民國紀年

## 三、備忘錄（MOU）

### 結構
比合約書簡化，通常用於合作意向的框架性約定。

```
合作備忘錄

○○○（以下簡稱「甲方」）與○○○（以下簡稱「乙方」），就○○○○合作事宜，達成以下共識：

一、合作目的
○○○○○○

二、合作範圍
○○○○○○

三、各方職責
（一）甲方：……
（二）乙方：……

四、合作期間
自簽署日起至中華民國000年00月00日止。

五、其他事項
○○○○○○

甲方：○○○（簽名蓋章）
乙方：○○○（簽名蓋章）

中華民國000年00月00日
```

### 撰寫要領
- MOU 通常不具強制法律效力，但仍應嚴謹
- 明確標示是否具法律拘束力
- 如需具法律效力，應升級為正式合約

## 五、委託書 / 授權書

### 結構

```
委託書

委託人：○○○
身分證字號：
地址：

受託人：○○○
身分證字號：
地址：

茲委託上列受託人代為辦理下列事項：

一、○○○○○○
二、○○○○○○

委託期間：自中華民國000年00月00日起至000年00月00日止。
委託範圍：（全權委託 / 限定委託事項）

委託人：○○○（簽名蓋章）
受託人：○○○（簽名蓋章）

中華民國000年00月00日
```

## 共通注意事項

1. **日期**：使用中華民國紀年
2. **金額**：重要金額同時以中文大寫和阿拉伯數字書寫，如「新臺幣壹拾萬元整（NT$100,000）」
3. **簽名蓋章**：法律文件以親自簽名並蓋章為原則
4. **份數**：註明一式幾份、各方各執幾份
5. **騎縫章**：多頁文件應蓋騎縫章
6. **見證人**：重要文件可加見證人簽名
7. **公證**：如需更高法律效力，可至法院或民間公證人處公證
