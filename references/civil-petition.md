# 人民對政府文書撰寫規範

本文件涵蓋一般民眾向政府機關提出申請、陳情、訴願等文書的格式與撰寫規範。

## 共通原則

人民對政府機關的文書，雖不受《公文程式條例》嚴格約束，但仍應注意：
- **格式整齊**：有基本結構，不宜用隨意信件格式
- **事實明確**：時間、地點、人物、經過要具體
- **訴求清楚**：明確說明希望機關做什麼
- **法令引用**：如能引用相關法規依據，更有說服力
- **禮貌得體**：對機關稱「貴」（如貴局、貴處），自稱本人姓名

## 一、陳情書

### 使用時機
人民對政府施政、行政措施或公共事務有意見、建議或不滿時，向機關反映。

### 結構

```
陳情書

受文機關：○○○○○○（如：○○市政府、○○部）

陳情人：
姓名：○○○
地址：○○市○○區○○路○○號
電話：0900-000-000
電子信箱：○○○@○○○.com（選填）

主旨：為○○○○○○一事，懇請貴（機關簡稱）惠予處理。

說明：

一、事實經過：
    ○○○○○○○○○○（具體描述問題發生的時間、地點、經過）

二、所受影響：
    ○○○○○○○○○○（說明此事對陳情人或公眾造成的影響）

三、相關依據：（選用）
    依據○○法第○條規定，○○○○○○

四、陳情訴求：
    懇請貴（機關簡稱）○○○○○○（具體說明希望機關採取的措施）

五、附件：（選用）
    1. ○○○○○○（照片、文件、證據等）
    2. ○○○○○○

謹呈
○○○○○○（受文機關全銜）

陳情人：○○○（簽名蓋章）

中華民國000年00月00日
```

### 撰寫要領
- **基本識別以真實姓名、地址為主**：符合《行政程序法》陳情章的基本要求
- **最少必要個資**：除法規或受理機關制式表單另有要求外，不預設要求身分證字號、身分證影本等敏感資料
- **事實具體**：避免籠統描述，要有時間、地點、頻率等具體資訊
- **訴求合理**：請求事項應在機關權責範圍內
- **語氣理性**：即使不滿，也應以理性方式陳述，避免情緒性用語
- **附件佐證**：如有照片、錄音、文件等證據，一併附上
- **送達方式**：可親送、郵寄或透過機關網站陳情系統
- 依《行政程序法》第171條，機關認為陳情有理由者應採取適當措施；認為無理由者，應通知陳情人並說明其意旨
- 依《行政程序法》第173條，無具體內容或未具真實姓名或住址之陳情，機關得不予處理

### 常用起首語
- 「為○○○○一事，懇請貴局惠予處理」
- 「關於○○○○問題，特此陳情」
- 「有關○○○○事宜，敬請貴府關注」

## 二、申請書

### 使用時機
人民依法規向政府機關申請許可、核准、發證等行政處分。

### 結構

```
○○○○申請書

受理機關：○○○○○○

申請人：
姓名：○○○
身分證字號：A000000000
出生日期：中華民國○○年○○月○○日
地址：○○市○○區○○路○○號
電話：0900-000-000

申請事項：○○○○○○○○○○

申請依據：依據○○法第○條（第○項第○款）規定

申請說明：
一、○○○○○○○○
二、○○○○○○○○

檢附文件：
一、○○○○影本○份
二、○○○○正本○份
三、○○○○○份

申請人：○○○（簽名蓋章）

中華民國000年00月00日
```

### 撰寫要領
- **查明法規依據**：先確認申請的法令依據及應備文件
- **文件齊全**：依規定備齊所有應附文件
- **資料正確**：個人基本資料務必正確
- **許多申請有制式表單**：先向機關索取或至網站下載，有制式表單的優先使用
- 依《行政程序法》第35條，申請書不合法定程式的，機關應通知補正

## 三、訴願書

### 使用時機
人民對行政機關的行政處分不服時，向上級機關或原處分機關提起訴願。

### 法定期限
**收到行政處分書之次日起30日內**提起（依訴願法第14條）。

### 結構（依訴願法第56條規定）

```
訴願書

訴願人：
姓名：○○○
出生日期：中華民國○○年○○月○○日
身分證字號：A000000000
住居所地址：○○市○○區○○路○○號
電話：0900-000-000
（如有訴願代理人，另載明代理人姓名、地址）

原處分機關：○○○○○○（作成行政處分的機關）

訴願事項：

一、原處分：
    ○○機關於中華民國000年00月00日以○○字第0000000000號
    函（或處分書），○○○○○○○○（處分內容）。

二、收受或知悉原處分之年月日：
    中華民國000年00月00日

三、訴願之聲明：
    請求撤銷（或變更）原處分。

四、訴願之事實及理由：

  （一）事實：
        ○○○○○○○○○○（敘述案件始末）

  （二）原處分違法或不當之理由：
        1. ○○○○○○○○
        2. ○○○○○○○○

  （三）法令依據：
        依○○法第○條規定，○○○○○○
        依行政程序法第○條規定，○○○○○○

五、附件及證據：
  1. 原處分書影本1份
  2. ○○○○○份
  3. ○○○○○份

謹呈
○○○○○○（訴願管轄機關，通常為原處分機關的上級機關）

訴願人：○○○（簽名蓋章）

中華民國000年00月00日
```

### 撰寫要領
- **法定期限**：收到處分書次日起30日內，逾期不受理
- **必要記載事項**（訴願法第56條）：訴願人姓名、出生年月日、住居所、身分證明文件字號；原行政處分機關；訴願請求事項；事實及理由；收受或知悉行政處分日期；受理訴願機關；證據；年月日
- **聲明要明確**：撤銷原處分、變更原處分、或確認原處分違法
- **理由要充分**：逐點說明原處分為何違法或不當，引用具體法條
- **經由原處分機關提起**：訴願書可經由原處分機關向訴願管轄機關提起（訴願法第58條）
- **附原處分書影本**：必附

### 訴願管轄機關
- 不服鄉鎮市公所處分 → 向縣市政府訴願
- 不服縣市政府處分 → 向中央主管部會訴願
- 不服中央各部會處分 → 向行政院訴願

## 四、異議書

### 使用時機
對政府採購決定、招標文件、評選結果等提出異議。

### 結構

```
異議書

受文機關：○○○○○○（招標機關）

異議人：
名稱：○○○○有限公司
統一編號：00000000
代表人：○○○
地址：○○市○○區○○路○○號
電話：（00）0000-0000

異議事項：
就貴機關○○○○招標案（案號：○○○○）之○○○○，提出異議。

異議理由：
一、○○○○○○○○
二、○○○○○○○○
三、依政府採購法第○條規定，○○○○○○

請求事項：
請貴機關○○○○○○

附件：
一、○○○○○份
二、○○○○○份

異議人：○○○○有限公司（蓋公司大小章）
代表人：○○○

中華民國000年00月00日
```

### 注意事項
- 異議須於規定期限內提出（依政府採購法規定）
- 異議不成，可於收到異議處理結果次日起15日內向採購申訴審議委員會申訴

## 五、申覆書

### 使用時機
對機關之答覆或處理結果仍不服時，再次表達意見。

### 結構
比照陳情書格式，但應明確引述前次陳情的機關回覆文號及內容，並說明不服的理由。

## 共通格式提醒

1. **用紙**：A4 白色紙張
2. **字體**：標楷體或新細明體，12～14pt
3. **日期**：使用中華民國紀年
4. **送達**：重要文書建議以雙掛號郵寄或親送並取得收件證明
5. **影本留存**：自行保留文書影本及郵寄憑證
6. **署名蓋章**：親自簽名，如有印章一併蓋章
7. **法人申請**：應蓋公司大小章（公司章＋代表人章）
